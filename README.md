# Counter App

This is a simple counter application built using React.  
It allows users to increase, decrease, and reset a value with a clean and responsive UI.

## Tech Stack
- React (useState)
- Tailwind CSS
- Vite

## Features
- Increment & Decrement functionality
- Reset button
- Responsive UI design

## Folder Structure
src/
 ├── Component/
 │   ├── CounterDisplay.jsx
 │   ├── CounterButton.jsx
 ├── App.jsx
